# ML_MODEL_COMPARISON
# 📊 Model Comparison (ML Assignment)

## 📌 Objective

Compare the performance of:

* Logistic Regression
* KNN
* Decision Tree

using the Iris dataset.

---

## 📂 Dataset

* Iris dataset (classification)
* 3 classes: Setosa, Versicolor, Virginica

---

## ⚙️ Tools Used

* Python
* Scikit-learn
* Matplotlib

---

## 📈 Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 0.97     |
| KNN                 | 1.00     |
| Decision Tree       | 1.00     |

---

## 📊 Visualizations

* Accuracy comparison graph
* Confusion matrix
* Accuracy trend graph

---

## 🧠 Conclusion

KNN and Decision Tree performed best due to their ability to capture non-linear patterns, while Logistic Regression showed slightly lower accuracy.
